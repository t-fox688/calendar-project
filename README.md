# 📅 Calendar App (가제)

## 📖 프로젝트 소개
Flutter와 Firebase를 활용해 개발하는 **개인 일정 관리 iOS/Android 앱**입니다.
방학 기간(2개월) 동안 기획부터 배포까지 전 과정을 직접 수행하며, 개인 포트폴리오 및 실제 스토어 배포를 목표로 합니다.

## 🛠️ 기술 스택 (Tech Stack)
- **Framework:** Flutter (Dart)
- **Backend:** Firebase (Firestore, Auth)
- **IDE:** Android Studio / Xcode
- **Platform:** iOS, Android

## 🎯 주요 기능 (Features)
- [ ] **캘린더 뷰:** 월간(Month) 달력 UI 제공 및 날짜별 일정 표시
- [ ] **일정 관리 (CRUD):** 일정 추가, 조회, 수정, 삭제 기능
- [ ] **데이터 연동:** Firebase Firestore를 이용한 실시간 데이터 동기화
- [ ] **카테고리 분류:** 일정 성격에 따른 색상 마커 표시

## 📅 개발 일정 (Roadmap)
- **1~2주 차:** 기획, 환경 설정, UI 프로토타입 구현 (완료)
- **3~4주 차:** 캘린더 핵심 로직 및 CRUD 구현
- **5~6주 차:** Firebase 연동 및 데이터 처리
- **7~8주 차:** 버그 수정, 문서화 및 앱 배포
